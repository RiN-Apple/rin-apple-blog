## df ~ディスクの使用量~

```bash
$ df

# -hオプションを付けると読みやすくなる(K,M,G表示)
$ df -h
```

## du ~ファイルのディスク使用量~

```bash
# カレントディレクトリの場合
$ du ./

# 集計するディレクトリの深さを１に
$ du -d 1 ./

# dfと同じく-hオプションも付けられる
$ du -h ./

# sortも加えたよく使う構成
$ du -d 1 . | sort -nr
```
