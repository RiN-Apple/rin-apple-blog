# RiN5-BLOG

RiN5さんのブログテンプレート。SSGしてます。

## 起動

```bash
yarn
yarn dev
```

<http://localhost:3000>を開く

## 利用した技術

- Next.js(SSG)
  - TypeScript
  - React
- prettier(コード整形)
- eslint(構文チェック)
- tailwind css(decoration)
- font awesome (アイコン)
- MDX(マークダウンでjsx)
- tocbot(目次)
- remark-prism(コードのハイライト)
- Firebase Hosting(デプロイ先)
